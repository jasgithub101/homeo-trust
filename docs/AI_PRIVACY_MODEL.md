# AI Privacy Model

AI and Explore must use de-identified records only.

Never expose:
- patient name
- phone
- email
- exact address
- emergency contact
- exact patient ID
- exact case ID
- doctor name
- raw reports/photos

Allowed:
- age range
- gender
- broad location if safe
- symptoms
- issue summary
- medicine
- potency
- dosage/frequency pattern
- patient condition trend
- improvement trend

AI output must include a decision-support disclaimer.
