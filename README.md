# Homeo Trust

A privacy-sensitive patient-management system for a single homeopathy clinic.
Self-hosted (single-tenant), with dynamic role/permission access control,
server-side authorization on every write, audit logging, and a de-identified
research surface (Explore). Built with Next.js (App Router), TypeScript,
PostgreSQL, and Prisma.

See `docs/PROJECT_WALKTHROUGH.md` for an end-to-end tour of the system, and the
rest of `docs/` for the product, data, security, and privacy models.
